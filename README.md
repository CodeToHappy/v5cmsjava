## v5cms项目
v5cms项目是使用java开发的多站点CMS系统，该系统使用MySQL数据库，开发使用Maven搭建，二次开发和使用都简单易用。
## 使用技术
**后端技术：**
- Spring
- SpringMVC
- MyBatis
- Shiro
- FreeMarker

**前端技术：**
- Bootstarp
- ace
- ueditor
- webuploader
